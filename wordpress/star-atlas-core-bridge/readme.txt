Generic WordPress bridge for Star Atlas Core.

== Description ==

Star Atlas Core Bridge is a single generic plugin for Star Atlas Media WordPress sites. It is configured by a Core-generated setup token and receives site-specific paths, push group, manifest, service worker, SDK, and update metadata from Core.

This skeleton does not implement full SSO, comments, push preferences, push sending, or auto-update policy logic.

== Installation ==

Install the plugin directory, activate it, open the Star Atlas Core admin page, and paste a setup token generated by Core.
